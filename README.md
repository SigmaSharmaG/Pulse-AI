# 🚀 PulseAI Agent – Intelligent Health & Fitness Coach

PulseAI is an AI-powered personal health and fitness coaching platform that provides **personalized workout plans, diet recommendations, and real-time conversational guidance** using modern AI technologies.

---

## 🧠 Problem Statement

Most fitness apps provide **generic plans** that fail to adapt to individual needs, progress, and lifestyle. This leads to:
- ❌ Poor results
- ❌ Lack of motivation
- ❌ Inconsistent routines

---

## 💡 Solution

PulseAI uses **Agentic AI + LLMs** to deliver:
- 🏋️ Personalized workout plans
- 🍽️ Smart diet recommendations
- 💬 AI chat-based coaching
- 📊 Progress tracking & adaptive feedback

---

## ✨ Features

### 🔹 Core Features
- User Authentication (Login / Signup)
- Personalized Workout Generator
- AI Diet Planner
- AI Chat Coach (real-time interaction)
- Progress Tracking System

### 🔹 Smart Features
- Feedback-based plan adjustment
- Context-aware AI responses
- Quick fitness suggestions
- Chat history storage (MongoDB)

### 🔹 UI/UX
- WhatsApp-style chat interface
- Clean & modern dark theme
- Responsive design
